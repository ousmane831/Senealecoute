# Séné à l'Écoute Services — React + Vite SPA

Copie SPA pure (sans SSR ni TanStack Start) du site, prête pour Netlify.

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Le dossier `dist/` est prêt à être déployé.

## Déploiement Netlify

- Build command : `npm run build`
- Publish directory : `dist`
- (Le fichier `netlify.toml` est déjà inclus.)
